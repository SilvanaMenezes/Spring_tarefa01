Descrição do Projeto

Esse projeto tem o objetivo de utilizar as rotas de uma aplicação Spring Boot para coletar e processar dados com a linguagem de programação Java, através da url do navegador.

Rotas do Projeto

Saudação ao Usuário

Mostra uma mensagem de boas vindas ao usuário e conta a quantidade de letras que tem no nome e sobrenome digitados.

    Para receber a mensagem, basta seguir os passos abaixo:

  1. Digite na Barra de Busca:

    localhost:4500/saudacao/{seuNome}_{seuSobrenome}

    Exemplo:

    digite: localhost:4500/saudacao/Fulano/Camargo

    Obs.: Utilize ( _ )underline para separar ambos os valores. Por favor

  2. Pressione ENTER

Calcular a média de duas Provas

Calculo da média de duas provas bimestais para saber a nota final de um aluno no semestre.

    Para Calcular a média das provas basta seguir os passos abaixo:

  1. Digite na Barra de Busca:

    localhost:4500/media/EscrevaValorDaProva1/EscrevaValorDaProva2

    Exemplo:

    Para a prova1: 6.4 e prova2: 7.5 , digite: localhost:4500/media/6.4/7.5

    Obs.: Para digitar o valor da prova1 e da prova2, utilize o ponto (.) em vez da vírgula (,) e ( / ) para separar ambos os valores. Por favor

  2. Pressione ENTER

Calcular IMC

A classificação do índice de massa corporal (IMC) pode ajudar a identificar problemas de obesidade ou desnutrição, em crianças, adolescentes, adultos e idosos.

    Para Calcular o seu IMC basta seguir os passos abaixo:

  1. Digite na Barra de Busca:

    localhost:4500/imc/peso=EscrevaSeuPeso&altura=EscrevaSuaAltura

    Exemplo:

    Para um peso 50.1 kg e altura 1.63 m , digite: localhost:4500/imc/peso=50.1&altura=1.63

    Obs.: Para digitar altura e peso, utilize o ponto (.) em vez da vírgula (,). Por favor

  2. Pressione ENTER
